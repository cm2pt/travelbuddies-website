# TravelBuddies Landing Page

Premium, high-converting landing page for “TravelBuddies | Organizamos Viagens em Família”.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

- Any static host that supports Vite output.
- Build with `npm run build` and deploy the `dist/` folder.

## Notes

- Language toggle persists in `localStorage`.
- Diagnosis form generates a message, copies to clipboard, and provides WhatsApp/Email/Instagram actions.
